- 🎨 Design sombre moderne avec thème cyan/violet

## 🛠️ Installation & Lancement

### Prérequis
- **Node.js** v14 ou supérieur
- **npm** v6 ou supérieur
- **Angular CLI 13** : `npm install -g @angular/cli@13`

### Étapes

```bash
# 1. Aller dans le dossier du projet
cd portfolio-jordan

# 2. Installer les dépendances
npm install

# 3. Lancer le serveur de développement
ng serve

# 4. Ouvrir dans le navigateur
# http://localhost:4200
```

### Build pour la production

```bash
ng build --configuration production
# Les fichiers seront dans dist/portfolio-jordan/
```

## 🎨 Personnalisation

- **Données** : Modifier `/src/app/services/portfolio.service.ts`
- **Couleurs** : Modifier les variables CSS dans `/src/styles.scss`
- **Photo** : Remplacer `/src/assets/images/jordan.jpg`
